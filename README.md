# HandGestureControl

Use [Mediapipe Hand] Gesture and [Simple Gradient Boosting Classifier]

Usage:
``python app.py --device=0 --width=960 --height=540 --min_detection_confidence=0.7 --min_tracking_confidence=0.5``
